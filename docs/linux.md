# Linux

Linux is not a supported release target for `0.1.0`.

This document exists so platform-specific documentation has a stable location as
the project grows.

## Current status

- No Linux executable is published for `0.1.0`
- Linux-specific testing and packaging are not part of this release
- Behavior may work during development, but it is not yet a supported platform

When Linux support is added, this document should cover build steps, runtime
notes, and packaging details for that platform.
